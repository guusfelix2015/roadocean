# 💻Iniciando o projeto

Para inciar o projeto crie um package.json

```bash
npm init -v (-V para que ele crie automaticamente sem fazer perguntas )
```

# 💻 Pacotes utilizados

- Nodemon
- Nunjucks
- Express

```bash
npm install nodemon -D (-D dependencias de desenvolvimento)
npm install express
npm install nunjucks (Template Engine)
```
